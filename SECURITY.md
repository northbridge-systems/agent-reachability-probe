# Security policy

## Reporting a vulnerability

If you discover a security issue in `agent-reachability-probe`, please report it privately by email to:

**security@northbridgesystems.de**

Please include:

- A description of the issue and its potential impact
- Steps to reproduce
- The version or commit hash affected

We aim to acknowledge reports within five working days. Please do not open a public issue for security-relevant findings until a fix is available.

## Scope

This policy covers the code in this repository. For vulnerabilities in upstream dependencies (Python standard library), please report them to the upstream project directly.
